# SmartMate 功能测试记录

> 测试日期：2026-07-12

> 测试方法：手工功能测试，Debug 构建
> 测试结果：**107/107 全部通过** ✅

---

## 本轮新增功能概要

1. **归档后永久删除**：已归档任务支持物理删除，连带清理全部依赖边，操作不可撤销
2. **逾期提醒**：截止时间已过的任务在卡片和聚焦槽中显示红色「已逾期」徽标和提示文案
3. **编辑权限收紧**：只有「待办」状态的任务可以编辑；InProgress/Done/Cancelled 需先 Redo 回 Todo 才能编辑
4. **QML 组件拆分**：TaskCard、TaskFocusCard、TaskFilterBar 独立为组件文件（代码质量提升，无功能变化）

| 8 | 删除后列表刷新 | 归档视图有 3 个任务 → 永久删除 1 个 | 列表自动刷新，计数变为 2 | ✅ |

## 测试结论

本轮新增的归档永久删除、逾期提醒、编辑权限收紧三项功能全部通过测试。删除操作的三层防御（UI 可见性 + Service 守卫 + SQLite 条件删除）均正确执行；逾期提醒仅对 Todo/InProgress 生效，完成/取消后自动消失；编辑权限精准限制在 Todo 状态，非 Todo 任务在 UI 菜单中直接隐藏编辑入口，Service 层双向守卫拒绝非 Todo 任务的编辑。上次 73 项全量回归无退化。
