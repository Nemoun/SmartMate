# SmartMate 功能测试记录

> 测试日期：2026-07-14
> 测试版本：main 分支（Qt Widgets 重构完成）
> 测试方法：手工功能测试，Debug 构建
> 测试结果：**112/114 通过，2 项不通过**
> 重构范围：View 层从 Qt Quick/QML 迁移至 Qt Widgets，Model 层不变，ViewModel 层引入 Contract 抽象

---

## 重构概要

| 维度 | 重构前 (QML) | 重构后 (Widgets) |
|------|-------------|-----------------|
| UI 框架 | Qt Quick / QML | Qt Widgets |
| 任务列表 | QML ListView + TaskCard.qml | QListView + TaskItemDelegate（自绘卡片） |
| 依赖图 | QML Flickable + Shape | QGraphicsView + QGraphicsObject |
| 拖拽 | QML Drag.active | QDrag（原生 MIME 拖拽） |
| 数据绑定 | QML 属性绑定 | WidgetBinding + signal/slot |
| 卡片交互 | TapHandler + ⋯ 按钮 | ⋯ 按钮 + 主次按钮区域 hit-test |
| 主题 | QML AppearanceTheme | WidgetTheme + QPalette 注入 |
